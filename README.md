# Customer-Segmentation---DS
Customer Personality Analysis using Data Science techniques to segment customers based on behavior, demographics, and spending patterns. Includes EDA, clustering, and insights to help businesses target the right audience and improve marketing strategies.

Project Overview
This project focuses on analyzing customer data to understand purchasing behavior and segment customers into different groups using machine learning techniques.

Objective
To identify different customer segments based on demographics, income, and spending patterns to help businesses improve marketing strategies.

Dataset
The dataset includes:
- Customer demographics (age, education, marital status)
- Income details
- Spending on products (wine, fruits, meat, etc.)
- Purchase behavior (web, store, catalog)
- Campaign responses

Exploratory Data Analysis (EDA)
- Analyzed customer behavior and spending patterns
- Identified correlations between income and spending
- Handled missing values and outliers

Model Used
- K-Means Clustering
- Elbow Method used to find optimal clusters
- Segmented customers into 4 groups

Key Insights
- Higher income customers spend more
- Customers without children spend more
- Identified high-value and potential customers

Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

Project Structure
- Data preprocessing
- EDA
- Model building
- Evaluation
 
Conclusion
The project successfully segments customers into meaningful groups, helping businesses target the right audience and improve decision-making.
